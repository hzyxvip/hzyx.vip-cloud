## 二、如何启动系统（本机）

需要同时运行**前台**和**后台**（可由 Cursor 协助执行）：

| 项目 | 目录 | 命令 | 访问地址 |
|------|------|------|----------|
| 前台 | 项目根目录 | `npm run dev` | 一般为 http://localhost:5173 或 5174 |
| 后台 | `backend` 目录 | `npm run dev` | 一般为 http://localhost:3006 |

**注意**：请固定使用同一个前台端口，否则浏览器里的数据不共享。
